# Neurosymbolic AI for Legal University Document Analysis
## Project Description
This project uses a neurosymbolic AI approach.

PDF documents (statute and regulation) are processed to extract articles into JSON format.  
The data is then reorganized and passed to a Large Language Model (LLM) to generate logical facts, which are used in an ASP program.

### Data Structure

- `data/raw/`  
  Contains the original PDF documents:
  - statute
  - regulation

- `data/processed/`  
  Contains the processed data:
  - extracted articles in JSON format

- `src/`  
  Contains all Python (`.py`) files used in the pipeline

- `output/`  
  Contains the logical facts generated by the LLM, ready to be used by the ASP program

- `asp/`  
  Contains the ASP rules and logic (`program.asp`)

### Workflow

1. Load PDF files from `data/raw/`
2. Extract and parse articles
3. Save structured data into `data/processed/articles.json`
4. Reorganize data into article pairs and save them in `data/processed/articles.csv`
5. Pass the pairs to an LLM to generate logical facts
6. Save the generated facts in the `output/` folder, ready to be used by `program.asp`
